# Onyx Web Application [Link](https://onyx-solutions.azurewebsites.net)

## Work in progress

## Description:

Full stack website for a service company (ie plumbing, electrician, etc) to manage customers, invoices, and job schedules. 

## Technologies used:

     - C# Dotnet Core 8
     - Angular Framework 17
     - TypeScript
     - Microsoft Identity
     - Entity Framework
     - SQL Server
     - Deployed to Azure

Now live on Azure!💻
